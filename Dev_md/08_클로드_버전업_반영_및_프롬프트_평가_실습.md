# 08. Claude 버전업 반영 & 프롬프트 평가 실습 추가

- **날짜**: 2026-06-18
- **목적**: 7시간 Claude 전용 강의 교재로 활용하기 위한 전체 콘텐츠 최신화 + 실습 메뉴 신설

## 1. 모델 라인업 전면 갱신

옛 라인업(Opus 4 / Sonnet 4 / Haiku 3.5, 200K, $15/$75 등)을 현재 기준으로 교체.

| 모델 | 모델 ID | 컨텍스트 | 입력/출력 (MTok) | 최대 출력 |
|------|---------|----------|------------------|-----------|
| Claude Fable 5 | `claude-fable-5` | 1M | $10 / $50 | 128K |
| Claude Opus 4.8 | `claude-opus-4-8` | 1M | $5 / $25 | 128K |
| Claude Sonnet 4.6 | `claude-sonnet-4-6` | 1M | $3 / $15 | 64K |
| Claude Haiku 4.5 | `claude-haiku-4-5` | 200K | $1 / $5 | 64K |

- Opus·Sonnet·Fable은 **1M 컨텍스트를 표준 가격**으로 제공(장문 프리미엄 없음).
- Fable 5를 "가장 강력한 프리미엄 모델"로 신규 소개(badge 컬러 추가).
- 갱신 파일: `model-comparison/data/models.js`, `about/data/model-comparison.js`, `api-usage/data/pricing.js`, `api-usage/data/getting-started.js`, `about/data/team.js`, `glossary`, `ai-news`.

## 2. 옛 모델 ID 일괄 마이그레이션

코드 예시의 날짜 접미사 모델 ID를 전부 alias로 교체(총 108곳).

- `claude-sonnet-4-20250514` → `claude-sonnet-4-6` (90)
- `claude-opus-4-20250514` → `claude-opus-4-8` (16)
- `claude-haiku-3-5-20241022` → `claude-haiku-4-5` (2)

## 3. 핵심 개념 최신화 (확장적 사고 → 적응형 사고)

- `prompt-engineering/data/extended-thinking.js`를 **"적응형 사고 & Effort"**로 재구성.
  - 폐기된 `budget_tokens` 방식 → `thinking={"type":"adaptive"}` + `output_config={"effort": ...}`(low/medium/high/xhigh/max).
  - Opus 4.7/4.8/Fable 5에서 `budget_tokens`·`temperature`/`top_p`/`top_k`는 400 오류임을 명시.
  - prefill 폐기 → 구조화 출력(`output_config.format`)/`client.messages.parse()` 안내.
- `chain-of-thought.js` 비교표, `messages-api.js` 파라미터 표(샘플링 제거·구조화 출력) 갱신.
- 가격 캐싱 설명: 쓰기 1.25배(5분)/2배(1시간), 읽기 0.1배로 정정.
- 용어집(Glossary)에 적응형 사고·Effort·프롬프트 캐싱·구조화 출력·스킬·Managed Agents 추가, 컨텍스트 200K→1M 정정.

## 4. 신규 메뉴: 프롬프트 평가 실습 (`/prompt-evaluation`)

수업에서 바로 쓰는 **루브릭 기반 자가 평가 도구**.

- 6대 기준(역할·맥락·구체적 지시·출력 형식·제약·예시) × 0~3점 = 총 18점 채점.
- 점수 입력 시 등급(A~D)·진행바·약점별 맞춤 개선 제안 자동 표시.
- Before→After 개선 사례 3종(마케팅 카피/코드 리뷰/문서 요약) + 실습 과제 3종.
- 신규 파일: `pages/prompt-evaluation/PromptEvaluation.tsx`, `styles/prompt-evaluation.css`.
