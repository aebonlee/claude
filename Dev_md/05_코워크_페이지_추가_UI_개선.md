## 주요 변경 사항

### 1. Claude 코워크 가이드 페이지 (신규)

경로: `/claude-cowork`

기존 `ClaudeWorkGuide` 패턴(사이드바 목차 + 마크다운 콘텐츠)을 동일하게 따름.

**6개 섹션 구성:**

| 섹션 | 내용 |
|------|------|
| 코워크 개요 | 코워크 정의, 기존 방식 비교, 활용 시나리오 |
| 공유 프로젝트 협업 | Projects 기능, Custom Instructions, 팀 워크스페이스 |
| 외부 서비스 연동 | MCP 프로토콜, Slack/Notion/Google Calendar 연동 실전 |
| 코워크 워크플로우 | 개발 워크플로우, 비즈니스 워크플로우, 자동화 |
| 베스트 프랙티스 | 효과적인 코워크 원칙, 안티패턴, 체크리스트 |
| 엔터프라이즈 코워크 | Enterprise 도입 가이드, 요금제 비교, ROI 측정 사례 |

**변경 전:** 섹션별 독립 디자인 (히어���, 비전 카드, 팀 카드, 타임라인, FAQ 아코디언 등)
**변경 후:** 통일된 가이드 레이아웃 (왼쪽 사이드바 목차 + 오른쪽 마크다운 콘텐츠)

**6개 사이드바 항목:**

| 항목 | 내용 |
|------|------|

### 3. 상단 네비게이션 메뉴 변경

**변경 전:**
```
소개 | Claude Code | 프롬프트 가이드 | API 가이드 | 모델 비교 | AI 뉴스 | 커뮤니티 | [로그인] [회원가입]
```

**변경 후:**
```
About | Claude Code | Co-work | 프롬프트 가이드 | API 가이드 | AI 뉴스 | 커뮤니티 | [로그인]
```

- "소개" → "About"으로 메뉴명 변경
- "Co-work" 메뉴 추가
- "모델 비교" 메뉴 제거 → About 페이지 내 모델 비교 섹션으로 이동
- 로그인/회원가입 2개 버튼 → 로그인 1개 버튼으로 통합

### 4. 로그인 후 풍선 도움말 (Tooltip)

- 로그인 직후 컬러 테마 버튼, 다크/라이트 모드 전환 버튼에 풍선 도움말 표시
- 5초 후 자동 사라짐, `sessionStorage` 기반 세션당 1회만 표시

### 5. 가이드 테이블 스타일 추가

- `guide-pages.css`에 `.table-responsive` 테이블 스타일 추가
- 헤더 배경, 테두리, 호버 효과, 반응형 가로 스크롤 지원

## 기술 노트

- 코워크/About ��이지 모두 `lazy(() => import(...))` 패턴 코드 스플리팅 적용
