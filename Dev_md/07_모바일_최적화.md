## 주요 변경 사항

### 1. Navbar 모바일 최적화 (768px 이하)

- 로고 크기 축소 (22px → 18px)
- 액션 버튼 크기 축소 (36px → 32px)
- 컬러닷, 테마 토글, 언어 버튼 모두 컴팩트화
- 풍선 도움말(tooltip) 모바일에서 숨김 처리
- 모바일 메뉴 패딩, 링크 사이즈 최적화

### 2. 가이드 페이지 사이드바 → 가로 스크롤 탭

768px 이하에서 사이드바가 상단 가로 스크롤 탭으로 전환:

- `position: sticky`로 스크롤 시 상단 고정
- 아이콘 숨김, 텍스트만 표시
- 좌우 스크롤 가능 (터치 스와이프 지원)
- 스크롤바 숨김 처리

### 3. Hero 섹션 모바일 최적화

- 제목 크기: 56px → 32px (768px) → 26px (480px) → 22px (360px)
- CTA 버튼 세로 배치, 100% 너비
- 캐러셀 패딩 축소, 슬라이드 폰트 축소
- 배지 크기 축소

### 4. 그리드 레이아웃 단일 열 전환

- 피처 카드: 4열 → 1열
- 학습 경로 카드: 3열 → 1열
- 통계: 3열 → 1열
- 워크플로우: 4열 → 2열 (768px) → 1열 (480px)

### 5. 테이블 & 코드블록 풀스크린

- 테이블: 좌우 마진 -16px로 화면 가득 채움, 테두리 제거
- 코드블록: 동일하게 풀스크린, 폰트 12px로 축소

### 6. 커뮤니티 페이지 모바일 최적화

- 헤더 세로 배치
- 검색 필드 세로 배치
- 게시글 아이템 세로 배치
- 댓글 폼 세로 배치
- 글 제목/메타 폰트 축소

### 7. 푸터 모바일 최적화

- 3열 → 1열 전환
- 바로가기: 2열 유지 (768px) → 1열 (480px)
- 패딩/폰트 축소

### 8. Toast 알림 모바일 최적화

- 전체 너비 (좌우 16px 마진)
- 최소 너비 제거
- 폰트 축소

### 9. 터치 디바이스 최적화

- `hover: none` + `pointer: coarse` 미디어 쿼리로 터치 디바이스 감지
- hover 시 transform 효과 비활성화 (깜빡임 방지)
- 최소 탭 타겟 44px 이상 확보 (접근성 기준)

### 10. 노치 폰 대응

- `env(safe-area-inset-bottom)` 지원
- 푸터, 모바일 메뉴 하단에 safe area 패딩 추가
