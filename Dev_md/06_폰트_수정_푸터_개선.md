## 주요 변경 사항

### 1. 폰트 깨짐 수정

**원인**: `base.css`의 `body` font-family에 `'Noto Sans KR'`이 지정되어 있었으나, 실제 Google Fonts import가 없어서 폰트가 로드되지 않음.

**수정**: `index.html`의 `<head>`에 다음 추가:

```html
<link rel="preconnect" href="https://fonts.googleapis.com" />
<link rel="preconnect" href="https://fonts.gstatic.com" crossorigin />
<link href="https://fonts.googleapis.com/css2?family=Noto+Sans+KR:wght@400;500;600;700;800;900&display=swap" rel="stylesheet" />
```

- `preconnect`로 DNS 사전 연결하여 폰트 로딩 속도 개선
- 400~900 weight 전체 포함 (본문, 소제목, 대제목 등에서 사용)

### 2. 푸터 바로가기 2열 배치

**변경 전**: 바로가기 링크가 1열로 나열 (세로로 길게)
**변경 후**: 2열 grid 레이아웃으로 변경

```css
.footer-link-list {
    display: grid;
    grid-template-columns: 1fr 1fr;
    gap: 4px 24px;
}
```

현재 바로가기 8개 항목:
- About / Claude Code
- Claude Work / Claude Co-work
- 프롬프트 엔지니어링 / API 활용
- AI 용어사전 / 커뮤니티
