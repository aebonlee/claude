# Claude Master - 프로젝트 개요

## 소개
Claude Master는 Anthropic의 Claude AI 모든 기능을 학습할 수 있는 종합 학습 플랫폼입니다.
Claude Code, Claude Work, Prompt Engineering, API Usage, Agent SDK 등을 체계적으로 학습할 수 있습니다.

## 주요 기능
- **Claude Code 가이드**: Hooks, MCP Servers, Slash Commands, IDE 통합 등 9개 섹션
- **Claude Work 가이드**: Projects, Artifacts, Team Features 등 6개 섹션
- **프롬프트 엔지니어링**: 기초부터 고급까지 7개 섹션
- **API Usage 가이드**: Messages API, Streaming, Tool Use, Vision, Pricing 등 7개 섹션
- **Agent SDK 가이드**: Architecture, Tools, Orchestration 등 5개 섹션
- **모델 비교**: Claude 모델별 성능/가격 비교표
- **용어 사전**: 30+ AI/Claude 관련 용어
- **학습 로드맵**: 5단계 체계적 학습 경로
- **리소스**: 공식 문서, 도구, 커뮤니티 링크
- **프롬프트 연습장**: 템플릿 기반 프롬프트 작성 연습
- **프롬프트 갤러리**: 카테고리별 프롬프트 예시
- **AI 뉴스**: 최신 AI 뉴스 모음
- **커뮤니티 게시판**: 공지, 자료, 질문, 자유 게시판
- **관리자 대시보드**: 사용자/게시글/통계 관리
